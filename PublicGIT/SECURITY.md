# Security Policy

## Supported Versions

We only support the latest version of GitCommitter. Please ensure you are running the most recent release before reporting an issue.

## Reporting a Vulnerability

We take the security of GitCommitter seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Do not open a public issue.** Instead, please send an email to security@wiroxa.com with a detailed description of the vulnerability, including:

-   The nature of the vulnerability.
-   Steps to reproduce the issue.
-   Potential impact of the vulnerability.
-   Any suggested fixes or mitigations.

We will acknowledge your report as soon as possible and work with you to resolve the issue in a timely manner. We ask that you keep the details of the vulnerability confidential until we have had a chance to fix it and release an update.

Thank you for helping us keep GitCommitter secure!
